# Product Spec

## 概要

本アプリは、ローカルPC上の文書ファイルを対象にした **全文検索専用 Web アプリ** である。  
ファイル名検索は対象外とし、ファイル内容検索に特化する。

利用形態は以下を想定する。

- 通常はローカルPC上で利用
- 必要に応じて Tailscale VPN 経由で私的アクセス
- インターネット公開はしない

## 目的

- 文書内容から目的のファイルを高速に見つける
- Google風のシンプルなUIで検索できるようにする
- ローカル用途に適した軽量構成にする
- macOS / Windows 両対応にする

## 主要要件

- 全文検索専用
- 検索時に対象フォルダのフルパスを指定できる
- 検索時に階層数を指定できる
- 検索前に対象フォルダのインデックスを必要時のみ更新できる
- 差分更新できる
- 同一対象を短時間で連続検索する場合、一定時間は更新を省略できる
- 対象拡張子を UI から選択できる
- パスは OS 差異を吸収して正規化する
- ローカル専用または Tailscale 経由の私的アクセスを前提とする

## 技術方針

- Backend: Python + FastAPI
- Frontend: React + Vite
- Search: SQLite FTS5

## 対応ファイル予定

### 初期
- `.md`
- `.json`
- `.txt`

### 将来拡張
- `.pdf`
- `.docx`
- `.xlsx`
- `.pptx`
- `.msg`
- `.excalidraw`
- `.drawio`
- `.drawio.svg`

## 非対象

- ファイル名検索
- OCR
- 公開インターネット運用
- 一般公開向け認証・認可
- Elasticsearch 導入

## 補足

現行コードでは、事前に検索対象フォルダを登録する方式は採用していない。  
検索時に `フルパス + 階層数` を指定し、その組み合わせを内部キャッシュとして管理する。

現行コードでは、VPN 経由利用を想定して FastAPI / Vite ともに `0.0.0.0` bind で起動できる。  
フロントエンドは `mac-mini` を許可ホストに含めている。
